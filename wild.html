Wild-Goppa-code decoder

Here we make a simple Python script available which demonstrates how to decode wild Goppa codes. The algorithm is described in detail in the article Wild McEliece by Daniel J. Bernstein, Tanja Lange, and myself.



The decoder is in fact an alternant decoder. Run the script using Sage which is a free open-source mathematics software system.



Download http://www2.mat.dtu.dk/people/C.Peters/wild-decoder.sg



Run sage wild-decoder.sg or just copy/paste the script into a running Sage program.
